# Obsidian-AI-Proofreader
# 中文错别字 AI 检测 —— 让每一篇文章都经得起推敲

## 写作，不该被错别字拖累

你是否经历过这样的场景：

- 精心撰写的文章发布后，被读者指出「的地得」用错
- 重要文档提交前，才发现「登录」写成了「登陆」
- 长篇笔记写完后，错别字散落各处，逐字检查耗时费力

**现在，有了「中文错别字 AI 检测」插件，这些问题将成为历史。**

---

## 它是什么？

「中文错别字 AI 检测」是一款专为 Obsidian 用户打造的智能中文校对工具。

它基于先进的大语言模型，能够**理解上下文语境**，精准识别文本中的错别字、音近字、形近字误用，并提供**一键修复**功能。

无需切换软件，无需复制粘贴，在 Obsidian 内即可完成「检测 → 查看 → 修复」的完整流程。

---

## 它能为你做什么？

### 🔍 智能检测，精准定位

- **语境理解**：不只是简单的词库匹配，AI 能理解上下文，识别「在/再」、「的/地/得」、「做/作」等易混淆字
- **全面覆盖**：音近字、形近字、语境错误、常见笔误，一网打尽
- **实时反馈**：检测结果以卡片形式清晰展示，错误位置一目了然

### ⚡ 一键修复，高效校对

- **快速替换**：点击按钮，瞬间完成错误修正
- **批量处理**：支持「全部应用」，一键修复所有检测到的错误
- **灵活忽略**：专业术语、特殊用词可加入忽略词库，避免误报

### 🎯 非侵入设计，专注写作

- **手动触发**：只在需要时运行，不打断你的写作思路
- **侧边栏展示**：检测结果在右侧面板呈现，不占用编辑空间
- **轻量高效**：文本智能分段，优化 API 调用，节省 Token 消耗

### 🔧 高度可配置，灵活适配

- **多服务支持**：兼容 DeepSeek、智谱、月之暗面等所有 OpenAI 接口服务
- **自定义模型**：自由选择模型，平衡检测精度与成本
- **测试连接**：一键验证 API 配置，确保服务正常

## 快速上手

### 第一步：安装插件

下载插件包，.将插件文件夹复制到 Obsidian 仓库的 `.obsidian/plugins/` 目录下。

### 第二步：配置 API

填写你选择的 AI 服务 API Key 和端点地址，点击「测试连接」验证。

### 第三步：开始检测

打开文档，点击侧边栏的「开始检测」按钮，等待结果呈现。

### 第四步：一键修复

查看检测结果，点击「替换」按钮完成修正，或「全部应用」批量处理。
